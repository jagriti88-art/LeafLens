## ğŸ—ï¸ System Architecture
1.  **Image Upload:** User takes a photo of a diseased leaf via the React app.
2.  **Analysis:** The Node.js server forwards the image to the FastAPI AI Engine.
3.  **Diagnosis:** The CNN model processes the image and returns a disease label and confidence score.
4.  **Treatment:** Node.js sends the label to Google Gemini to generate an organic cure.
5.  **Persistence:** The diagnosis and cure are saved to the user's MongoDB history.
